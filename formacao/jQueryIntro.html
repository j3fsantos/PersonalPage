Starting jQuery

Reviewing CSS selectors

Semantics of selectors:

  • #id1 - selects the element whose id is id1

  • .class1 - selects the elements whose class is class1

  • [attr1] - selects the elements that have an attribute named attr1

  • [attr1=value1] - selects the elements that have an attribute named attr1 and map it to value1

  • A B - selects the elements that match B and are descendants of an element that mathches A

  • A < B - selects the elements that match B and are direct descendants of an element that matches A

  • A ~ B- selects the elements that match B and are siblings that come after an element that matches A

  • A + B- selects the elements that match B and that immediately follow an element that matches A

  • A, B - selects all the elements that match A and all the elements that match B (Group Selector)

The jQuery Function - 1

The jQuery functions returns a special object that represents a set of zero or more DOM elements - a jQuery object


The jQuery function can be invoked in 4 different ways

  1. You can pass it a CSS selector

  2. You can pass it an Element Node, a Document Node or the Window object

  3. You can pass it a string of HTML text

  4. You can pass it a function

The jQuery Function - 2

Invoking jQuery with a CSS selector - it returns the set of elements of the current document that match the selector

Optional Second argument: CONTEXT. It only returns the matching descendants of the specified element

The jQuery Function - 3

Invoking jQuery on an element node, document node, or window object - it returns the same element wrapped in a jQuery object

Why? To use the jQuery methods to manipulate the element instead of using raw DOM methods


Invoking jQuery on an html string - it creates the html element corresponding to the string and returns that element wrapped in a jQuery object.

Optional Second Argument: an array of properties that are to be assigned to the element


Invoking jQuery on a function: the function will be executed when the document has been loaded and the function is ready to be executed

Inside this function the this will be bound to the document object

jQuery objects

A jQuery object is a special objects that wraps a given set of DOM elements and provides extra functionality to manipulate them

  • A jQuery object is an array-like object in that you can access all the objects that it encapsulates using the brackets notation

  • If you need to obtain the real array of objects corresponding to that jQuery object just invoke the function toArray()

  • jQuery objects have the following special properties: selector, context, jquery

jQuery vocabulary

Terminology to remember:

  • jQuery function: the function that is bound to the variable jQuery (or $). It creates jQuery objects and register functions to be called when the DOM is ready.

  • jQuery object: an object returned by the jQuery function. A jQuery represents a set of document elements.

  • jQuery function: a function that is defined in the namespace of the jQuery function.

  • jQuery method: a method of a jQuery object

Getting and Setting CSS classes


To interact with CSS classes:

  • addClass(): adds the specified class to the selected elements

  • removeClass(): removes the specified class from the selected elements

  • toggleClass(): adds the specified class to the selected elements, if it is not already present

  • hasClass(): checked if the selected elements have a given class

Getting and Setting the Window Geometry - 1

offset(): the position of the selected element relative to the viewport. It returns an object with the left and top properties that hold the X and Y coordinates.


position(): getter only. It returns the element position relative to their offset parent.

Modifying the Document Structure - 1

$(target).method(content)

  • append(): Insert content at the end of the target

  • prepend(): Insert content at the start of the target

  • after(): Insert content after the target

  • before(): Insert content before the target

  • replaceWith(): Replace target with content

$(content).method(target)

  • appendTo(): Insert content at the end of the target

  • prependTo(): Insert content at the start of the target

  • insertAfter(): Insert content after the target

  • insertBefore(): Insert content before the target

  • replaceAll(): Replace target with content

Remark 1:In both cases, the methods return the target


Remark 2:In the first case the content can be an HTML string

DOM traversal

A jQuery object provides the following methods for DOM traversal

  • find(): receives as input a selector and outputs the a jQuery object that contains all the objects that match the selector

  • children(): selects the direct descendants of the first element

  • parent(): selects the direct parent

  • filter(): reduces the elements to those that match the selector given as input
